4. （中世纪）基尔瓦基斯瓦尼

1. 罗马斗兽场（前置：娱乐中心、竞技场、平原）：辐射六格市中心
2. 莫索拉斯王陵墓（前置：靠近港口的海岸）：大工程师次数+1
3. （文艺复兴）布达拉宫：加市政卡外交槽
4. （工业时代）大本钟：国库经济翻倍，+1经济政策卡
5. （工业时代）牛津大学：当前城市+20%科技，解锁2个科技

## 对局
### 远古时期
铺2城（保证一个城有沙漠平原）
首都先出：
1. 2侦察
2. 造2建造者（再买1个建造者）：造金字塔和神谕
3. 移民：敲纪念碑

不够黄金：先出阿玛尼宗主城邦

黄金：马格努斯

### 古典时期
远古2城：完成阿帕达纳宫和神谕

补充区域和生产建造者

有高相邻加成则先补学院，否则先补商业区（增加商路，一整局要10条商路）

选一个树多的地方拍城（森林、雨林、石头、鹿）：
1. 先建市政广场以及一级建筑祠堂
2. 派马格努斯过去砍树生产开拓者，铺4-6城（新城先敲纪念碑，有高相邻位先学院，无则先商业中心和港口）


规划一个城市圈中心城市，建造娱乐中心和一级建筑竞技场，之后建罗马斗兽场
找一个城市建港口和莫索拉斯王陵墓

### 中世纪-文艺复兴
1. 沿海城市：基尔瓦基斯瓦尼
2. 选三个及以上城市造工业区和建筑工作坊，作用是发电，尽量辐射到更多城市
3. 有学院的城市补大学
4. 造紫禁城

### 工业时代-现代
1. 发电城造好燃煤发电厂
2. 学院城造好实验室
3. 提前在树多的地方建2航天城
4. 大本钟和牛津能建就建
5. 用点了发包人的瑞纳在航天城买出航天中心（有余力可以买出工业区和军营其他建筑），派马格努斯过去砍树走项目，砍完树补上发电设施
6. 派平伽拉到另一个航天城走激光站项目
7. 其他城市全都生产建造者或者转学院项目
